# ChordLab 🎵

A guitar chord reference tool with interactive fretboard diagrams. Covers 35 chords across 6 categories with search, filtering, and detailed chord info. Open `index.html` in any browser — zero dependencies.

## Features
- 35 chords: Major, Minor, 7th, Sus, Diminished/Augmented, Power
- SVG fretboard diagrams rendered dynamically in JavaScript
- Search by chord name, type, or notes
- Filter by category
- Click-through modal with full chord detail (notes, fingering, mood)
- Random chord picker
- Zero dependencies — pure HTML, CSS, JavaScript

## Tech Stack
- Vanilla JavaScript (ES6+)
- Dynamic SVG generation
- HTML5 / CSS3

## How to Run
Open `index.html` in any browser. No server needed.

## Chord Categories
| Category | Examples |
|----------|---------|
| Major | C, D, E, F, G, A, B |
| Minor | Am, Em, Dm, Cm, Fm, Bm |
| 7th | G7, Cmaj7, E7, Amaj7 |
| Sus | Dsus2, Dsus4, Esus4, Gsus4 |
| Dim/Aug | Bdim, Caug, Edim |
| Power | E5, A5, D5, G5 |
